# tsconfig.json

## Usage

```bash
npm install --save-dev @cablanchard/tsconfig.json
```

In `tsconfig.json`

```json
{
  "extends": "@cablanchard/tsconfig.json",
  "include": [
    "lib/**/*"
  ]
}
```
